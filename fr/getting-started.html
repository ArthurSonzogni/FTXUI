Installer FTXUI

Pour configurer FTXUI dans votre projet, suivez le guide d'installation, qui fournit des instructions pour plusieurs systèmes de construction et gestionnaires de paquets.

Exemple Minimal

Enregistrez le code suivant sous le nom de main.cpp :

#include <ftxui/dom/elements.hpp>
#include <ftxui/screen/screen.hpp>
#include <iostream>
int main() {
using namespace ftxui;
Element document = hbox({
text("left") | border,
text("middle") | border | flex,
text("right") | border,
});
auto screen = Screen::Create(Dimension::Full(), Dimension::Fit(document));
Render(screen, document);
screen.Print();
}
